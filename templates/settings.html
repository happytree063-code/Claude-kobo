🔔 設定手機推播通知

每週四書單更新時，手機會自動收到通知。使用免費的 ntfy app 接收推播。

1
安裝 ntfy app（免費）
📱 App Store 🤖 Google Play
2
訂閱你的專屬頻道
打開 ntfy app → 右下角 + → 輸入下方你設定的頻道名稱 → Subscribe
3
在 Render 設定環境變數（見下方說明）
設定好後點「發送測試通知」確認能收到

⚙️ 在 Render 設定環境變數

因為環境變數要在伺服器上設定，請照以下步驟操作（只需要做一次）：

  1. 前往 render.com → 進入你的 kobo-99-deals 服務
  2. 點左側選單 Environment
  3. 點 Add Environment Variable
  4. Key 填 NTFY_TOPIC，Value 填你想要的頻道名稱（英文+數字，例如 kobo99-alice-2024）
  5. 點 Save Changes → Render 會自動重新部署
  6. 回到這個頁面，點「發送測試通知」確認

💡 頻道名稱是公開的，建議加一串隨機數字讓別人不容易猜到，例如 kobo99-alice-5829
